# Personal-Web

A personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

This runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Deployment with Vercel

This project is configured for deployment on Vercel.

### Prerequisites

Install the Vercel CLI globally:

```bash
npm i -g vercel@latest
```

### Check for Updates

To check if your Vercel projects require updates:

```bash
vercel project ls --update-required
```

### Deploy

```bash
vercel
```

For production deployment:

```bash
vercel --prod
```
